# OzNetNerd AWS Organisations

Convenience Python module for AWS Organisations & STS.

## Installation

```
pip3 install onnawsorgs
```

## Docs

Generate docs:

```
pip3 install pdoc3
PYTHONPATH=. pdoc --html orgs --force -o ../docs/onnawsorgs
```
